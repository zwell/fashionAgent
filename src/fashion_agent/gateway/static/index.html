🔄 自动上新 SOP 工作流
完整的新品上架流程：Design Agent → [Visual + Marketing 并行执行] → 人工审核 → 发布
工作流程图
① Design Agent (CoT) → 生成设计提案
② Visual Agent (ReAct) + Marketing Agent (ToT) — 并行执行
③ 审核节点 → 自动/人工审核
④ 聚合输出 → 完整上架包
🎨 设计提案
Design Agent (CoT) — 基于时尚趋势和参考商品，生成新品设计方案（色彩/材质/版型/细节）
📷 视觉生成
Visual Agent (ReAct) — 观察商品信息 → 制定拍摄策略 → 生成商品图/模特图/场景图 → 质量评估
✍️ 文案生成
Marketing Agent (ToT) — 为商品生成多种风格的文案，选择最优输出
📦 智能补货
SupplyChain Agent (ReAct) — 观察库存 → 预测需求 → 生成补货建议
🏷️ 清仓决策
SupplyChain + Marketing Agent — 综合库存、竞品、销量制定清仓策略
📈 趋势分析
Data Agent (CoT) — 分析指定季节的时尚趋势：色彩、材质、版型、图案
🎯 通用任务提交
直接调用 Master Agent — 自由组合 task_type, instruction, params
